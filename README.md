# 주요기능

## 다클라이언트 로그인

사용자가 거상을 다중으로 실행 시킬때, 실행 방법을 간소화 해줍니다.

# 시작하기

## 설치 및 실행 방법

1. [자바](https://www.java.com/ko/download/)를 설치해주세요.

2. [거상로더](https://github.com/joseongho1985/GersangLoader/releases)를 다운받아 압축해제 해주세요.

3. gersangloader.bat을 실행하거나, app.jar를 관리자 권한으로 실행합니다.

## 다클라이언트 로그인 순서

1. 상단 메뉴에서 설정 > 클라이언트를 선택 후, 각 클라이언트의 위치를 등록해주세요. 클라이언트 등록은 최초 한번만 등록하면 됩니다.

2. 상단 메뉴의 설정 > 로그인을 선택 후, 아이디와 비밀번호를 입력해주세요.

3. 오른쪽 로그인 단추를 눌러 거상 홈페이지에 로그인 되는 것을 확인해주세요.

4. 왼쪽 실행 버튼을 클릭합니다.
